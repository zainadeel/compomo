# CompoMo — `@ds-mo/ui`

Stencil web components (`<ds-*>`) styled with [TokoMo](https://github.com/zainadeel/TokoMo) design tokens. Stencil also generates **Angular** proxy directives and **React** wrappers from the same source.

## Install

```bash
npm install @ds-mo/ui @ds-mo/tokens @ds-mo/icons
```

**Required peer dependencies:**

- **`@ds-mo/tokens`** — CSS custom properties (colors, dimensions, typography, effects).
- **`@ds-mo/icons`** — runtime SVG catalog for `<ds-icon>` (resolved from your app's IcoMo install, not bundled inside `@ds-mo/ui`).

## Setup

Import TokoMo tokens globally (once, at your app root):

```ts
import '@ds-mo/tokens';
import '@ds-mo/tokens/reset';
import '@ds-mo/tokens/globals';
```

Register the custom elements you use (each module auto-defines its tag):

```ts
import '@ds-mo/ui/dist/components/ds-button-filled.js';
import '@ds-mo/ui/dist/components/ds-button-unfilled.js';
import '@ds-mo/ui/dist/components/ds-text.js';
```

**Angular:** `import { DsButtonFilled } from '@ds-mo/ui/angular'` (or module that re-exports proxies).

**React:** `import { DsButtonFilled, DsText } from '@ds-mo/ui/react'`.

See [framework integration](https://github.com/zainadeel/compomo/blob/main/docs/framework-integration.md) for shell/nav first-paint rules.

## Architecture

```
TokoMo (@ds-mo/tokens)  ← CSS variables, JSON, TypeScript; source is Figma
    ↓
IcoMo (@ds-mo/icons)    ← SVG catalog; peer dependency at app bundle time
    ↓
CompoMo (@ds-mo/ui)     ← Stencil <ds-*> + generated Angular/React wrappers
    ↓
Apps (Webapp, lab, …)
```

## Icon pattern

Use `<ds-icon name="ArrowRight" />` with IcoMo export keys, or slot an icon into `slot="icon"` on buttons and similar components.

## Token dependency

All styling uses TokoMo CSS custom properties. Components render unstyled if `@ds-mo/tokens` is not imported.
