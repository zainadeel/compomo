# Accessibility review

Use this page for the visual review that automated Axe checks cannot complete
reliably—especially component boundaries, state indicators, focus treatments,
and intentionally inactive content.

## How to review

<ol>
  <li>View the matrix at <strong>100% browser zoom</strong> on a representative display.</li>
  <li>Compare Light and Dark side by side.</li>
  <li>For every state, inspect the control boundary, state cue, text or icon, and focus treatment against the immediately adjacent color.</li>
  <li>Tick <strong>Reviewed</strong> in the matrix as you work. These ticks are temporary and intentionally reset when the page reloads.</li>
  <li>Use <strong>Open full story</strong> for interaction, keyboard testing, other sizes, and the Accessibility panel's <strong>Incomplete</strong> details.</li>
</ol>

## Contrast guide

<table>
  <thead>
    <tr>
      <th>What is being reviewed</th>
      <th>Minimum</th>
      <th>Notes</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Normal text, including meaningful placeholder text</td>
      <td>4.5:1</td>
      <td>Large text may use 3:1.</td>
    </tr>
    <tr>
      <td>Essential control boundary or icon</td>
      <td>3:1</td>
      <td>Applies when needed to identify the active control.</td>
    </tr>
    <tr>
      <td>Selected mark, radio dot, switch state, slider thumb</td>
      <td>3:1</td>
      <td>Compare against immediately adjacent colors.</td>
    </tr>
    <tr>
      <td>Focus indicator</td>
      <td>3:1</td>
      <td>It must remain visible in both themes and every supported surface.</td>
    </tr>
    <tr>
      <td>Genuinely inactive/disabled or purely decorative content</td>
      <td>Exempt</td>
      <td>The exemption does not apply to merely low-priority active content.</td>
    </tr>
  </tbody>
</table>

Button borders are conditional: a subtle border can be supplemental when the
button is already identifiable from its visible label, icon, fill, and context.
Checkbox, radio, input, and similar boundaries usually carry essential control
information and therefore need the 3:1 treatment.
